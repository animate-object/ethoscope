# Behavior Schema Tool

Ethoscope intends to help build the 'perfect tool' for cataloging the behavior of a specific animal (or group of animals) in a specific context.

To accomplish this, it lets you define a 'behavioral schema' based on your prior knowledge. The behavior schema is a list of behaviors you want to catalog. For instance, if observing a bird in a tree: "sing", "fly", might be behaviors.

Optionally for complex behaviors you can specify 'tags' which subcategorize the behavior. For the previous example you may choose sing => song 1 | song 2 | song 3, fly => hop branch | hop tree | far away.

Together these behaviors and tags will configure a focused purpose built tracker experience that aids in rapidly cataloging behaviors with limited screen engagement.

Each schema you create is versioned to support changes over time. The behavior tracker will always use the latest schema. When running analysis you can choose to include events from different schemas or schema ranges.
